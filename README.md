##  SAUTI — A Safe Space for Survivors

**SAUTI** is a digital sanctuary where survivors can share their stories, find community, and access support. Built with empathy, cultural grounding, and mobile-first design, SAUTI centers African voices and fosters healing through storytelling.

---

### Vision

SAUTI means voice in Swahili. This platform amplifies voices that have been silenced, creating space for dignity, resilience, and transformation.

---

###  Tech Stack

- **Frontend**: React + Vite + TailwindCSS + DaisyUI  
- **Routing**: React Router  
- **State Management**: Context API 
- **Version Control**: Git + GitHub  
- **Design System**: Synthwave + Base themes with custom overrides

###  Features

-  **Share Your Story**: A form for survivors to submit their experiences safely  
-  **Navigation**: Intuitive, mobile-friendly layout with clear routes  
-  **Visual Identity**: Banners, illustrations, and color schemes that evoke sanctuary and hope  
- **Community-first**: Designed for real people, with real healing in mind

---

###  Values

- **Healing**: Every page is designed to feel safe and gentle  
- **Dignity**: Survivors are honored, not exposed  
- **Community**: Built for and with African women  
- **Voice**: Stories are central, not secondary

---

### Getting Started

```bash
# Clone the repo
git clone https://github.com/Flora72/SAUTI.git

# Navigate to frontend
cd sauti/frontend

# Install dependencies
npm install

# Run the app
npm run dev
```
